# Bakery Public P&L

Public Rugpull Bakery leaderboard P&L dashboard for Vercel.

## Local Preview

```bash
npm run dev
```

Open `http://localhost:4890`.

## Deploy For Free On Vercel

1. Push this repository to GitHub.
2. Create a new Vercel project.
3. Set the project root directory to `public-bakery-pnl`.
4. Keep the default build settings. There is no build command.
5. Deploy.

The public app only exposes the Bakery P&L page and `/api/pnl-top100`. It does not include local bot controls or credentials.
